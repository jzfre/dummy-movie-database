# SPA Template



## Available Scripts
Firstly, do `npm install`, you need all those packages.
Then you need to set .env.local file. Take a look at **env.d.ts** file, you will need an OMDB api key.


```bash
$ npm run dev
```

```bash
$ npm run build
```

```bash
$ npm run preview
```

```bash
$ npm run lint
```

## Tech stack
- Vite + React TS
- Eslint
- Prettier
